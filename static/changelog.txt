Tree Editor Changelog
=====================

May 22, 2026
  - Renamed -par (paragraphs widget) to -sec (sections)

May 21, 2026
  - Added "shuffle children" node command (Alt+c r)
  - Added "flatten children" node command (Alt+c f)
  - Added "children to array" node command (Alt+c a)
  - Added "collapse children" node command (Alt+c c)
  - Added "expand children" node command (Alt+c x)
  - Added "expand only nodes up to level" global command (Control+DIGIT)
  - Added "expand only nodes up to level" node command (Control+Alt+DIGIT)
  - Added "delete nodes on the same level" node command (Alt+i d)
  - Added "expand all nodes on this level" node command (Alt+i x)
  - Added "collapse all nodes on this level" node command (Alt+i c)
  - Added -out widget (outline)
  - Changed Fonts
  - Moved default widget into widgets (-txt)
  - Allow widgets in "-pts" widget
  - Added Ctrl+Click selection for the widgets -opt, -rad, -tbs and -pgs
  - Added the ability to create a table of contents for the -par widget (via $toc)

May 20, 2026
  - Added changelog to help
  - Added "recover last session" global command (Alt+x r)
  - Added "delete siblings" node command (Alt+s d)
  - Added "sort children" node command (Alt+c s)
